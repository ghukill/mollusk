# mollusk
Mollusk

## Development

### Installation

1- Clone repository
```shell
git clone https://github.com/ghukill/mollusk
cd mollusk
```

2- Create virtual environment and install dependencies:
```shell
uv venv .venv
uv sync --dev
```

3- Run tests:
```shell
uv run pytest -vv
```

3- Run linting:
```shell
uv run ruff check src/
```
